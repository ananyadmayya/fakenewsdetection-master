# Fake News Detection using NLP and Deep Learning

This project focuses on detecting fake news using Natural Language Processing (NLP) and deep learning models.

## Project Structure

notebook/
Contains Jupyter notebooks used for data preprocessing and model training.

src/
Contains the main application code including classifier logic and the Streamlit interface.

models/
Contains the trained machine learning models.

requirements.txt
Lists all Python dependencies required for the project.

## Models Used

- Naive Bayes
- Word2Vec
- LSTM / GRU
- Transformer-based models

## Technologies Used

Python  
TensorFlow / Keras  
Scikit-learn  
NLTK  
Gensim  
Streamlit  
Transformers  

## Run the Application

Install dependencies:

pip install -r requirements.txt

Run the Streamlit app:

streamlit run src/streamlit_app.py
